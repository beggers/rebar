# User Asks

This directory is the supervisor-owned intake for future human notes.

- Put new notes in `ops/user_asks/inbox/`.
- Do not put `USER-ASK` notes into `ops/tasks/ready/`.
- The supervisor handles one `USER-ASK` at a time and archives processed notes into `ops/user_asks/done/`.
- Harness and operating-model asks belong here by default.
